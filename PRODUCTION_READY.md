## 🏃‍♂️ **PASOS PARA LANZAR HOY MISMO**

### **Paso 1: Configurar Stripe (15 minutos)**
```bash
# 1. Ve a https://stripe.com
# 2. Activa tu cuenta para pagos reales
# 3. Obtén las claves LIVE (pk_live_ y sk_live_)
# 4. Configura webhooks en: tu-dominio.com/api/webhooks/stripe
```

### **Paso 2: Configurar Emails (10 minutos)**
```bash
# 1. Ve a https://resend.com
# 2. Registra tu dominio
# 3. Obtén API key
# 4. Configura DNS records
```

### **Paso 3: Deploy (5 minutos)**
```bash
# Opción A: Netlify
git push origin main
# Conectar repo a Netlify + configurar variables de entorno

# Opción B: Vercel
vercel --prod
# Configurar variables de entorno en dashboard
```

### **Paso 4: Variables de Entorno**
```bash
# Copiar estas variables a tu hosting:
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_live_...
STRIPE_SECRET_KEY=sk_live_...
STRIPE_WEBHOOK_SECRET=whsec_...
RESEND_API_KEY=re_...
FROM_EMAIL=noreply@tudominio.com
SALES_EMAIL=ventas@tudominio.com
NEXT_PUBLIC_APP_URL=https://tudominio.com
ENABLE_EMAIL_NOTIFICATIONS=true
```
